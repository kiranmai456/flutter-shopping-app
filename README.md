# 🛍️ Flutter Shopping App

A modern e-commerce mobile application built with **Flutter** and **Dart** that enables users to browse products, view product details, manage a shopping cart, and enjoy a smooth shopping experience. The application follows a modular architecture and uses **Provider** for efficient state management.

---

## ✨ Features

* 🛒 Browse Products
* 📦 View Product Details
* ➕ Add Items to Cart
* ➖ Remove Items from Cart
* 💰 Automatic Cart Total Calculation
* 🔄 Provider State Management
* 🧩 Reusable Widgets
* 📱 Responsive UI
* ⚡ Smooth Navigation

---

## 🛠️ Tech Stack

* 💙 Flutter
* 🎯 Dart
* 📦 Provider (State Management)
* 🎨 Material Design

## 📂 Project Structure

```text
lib/
│
├── models/
├── providers/
├── screens/
├── widgets/
├── helpers/
├── assets/
└── main.dart
```

### Folder Description

* **models/** – Defines product and cart data models.
* **providers/** – Manages application state using Provider and ChangeNotifier.
* **screens/** – Contains application screens such as Home, Product Details, and Cart.
* **widgets/** – Reusable UI components like product cards and custom buttons.
* **helpers/** – Utility methods and helper functions.
* **assets/** – Images, icons, and other static resources.
* **main.dart** – Entry point of the application.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kiranmai456/flutter-shopping-app.git
```

### Navigate to the project

```bash
cd flutter-shopping-app
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
